# Features

Dokumen ini berisi daftar fitur utama untuk aplikasi KBT — Kosuka Bali Trip.
Fitur-fitur ini dikelola melalui sistem batch (FXX).

> **Catatan:** Project ini adalah project baru (greenfield). Fitur-fitur aplikasi di bawah ini masih berstatus **Planned** karena belum ada implementasi teknis.

## KBT Application Features (Planned)
- **Home / Landing Page**: Halaman utama pengenalan Kosuka Bali Trip.
- **About Kosuka Bali Trip**: Halaman informasi profil perusahaan/layanan.
- **Tour Packages**: Daftar paket wisata Bali yang ditawarkan.
- **Destination Highlights**: Highlight destinasi populer di Bali.
- **Gallery**: Galeri foto dan video aktivitas wisata.
- **Contact / WhatsApp Inquiry**: Integrasi kontak dan WhatsApp untuk pertanyaan langsung.
- **Booking Inquiry Form**: Formulir pemesanan/inquiry paket wisata.
- **SEO & Content Structure**: Optimasi SEO dan struktur konten yang baik.
- **Admin CMS**: (Optional/Future) Dashboard admin untuk mengelola paket dan konten.
- **Deployment / Domain Setup**: Persiapan environment production.

## Workflow Foundation Features (Completed / Internal)
- **Source of Truth System**: GitHub sebagai satu-satunya kebenaran mutlak.
- **Roomchat Role System**: Pemisahan tugas antara Manager, Reviewer, dan Eksekutor.
- **Batch Gate System**: Mekanisme validasi sebelum eksekusi dimulai.
- **Feature Batch Tracker**: Pelacakan fitur menggunakan kode FXX.
- **Specialist Analysis Room Workflow**: Alur kerja penggunaan spesialis eksternal.
- **History Layer Separation**: Pemisahan layer history dari workflow/control layer.
- **Persistent Project Memory Layer**: Folder docs/history/ sebagai penyimpan status dan riwayat permanen.
