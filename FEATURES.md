# Features - Kosuka Bali Trip

## 🚀 Fitur MVP (Minimum Viable Product)
- **Landing Page**: Menampilkan banner hero, paket populer, dan testimoni.
- **Daftar Paket Wisata**: Halaman berisi list destinasi atau paket tur yang tersedia.
- **Detail Paket Wisata**: Halaman spesifik yang menampilkan itinerary, harga, inklusi, dan eksklusi paket.
- **Form Pemesanan (WhatsApp Redirect)**: Fitur booking yang akan melempar (redirect) data isian user langsung ke WhatsApp admin.
- **Halaman Statis**: About Us dan Contact Us.

## ⏳ Fitur yang Ditunda (Non-MVP)
- Integrasi Payment Gateway (Midtrans/Stripe).
- Sistem Autentikasi (Login/Register User).
- Dashboard User (Melihat riwayat pemesanan).
- Sistem Multi-bahasa (Fokus pada bahasa Inggris/Indonesia terlebih dahulu).
- Content Management System (CMS) Admin Panel yang kompleks.
