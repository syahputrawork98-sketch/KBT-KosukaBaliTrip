# Kosuka Bali Trip

Kosuka Bali Trip adalah project website travel dan tour berbasis Bali.

Repository ini digunakan sebagai pusat pengembangan website, dokumentasi, dan alur kerja project.

## Project Status

Project sedang berada pada tahap awal.

Fokus saat ini:
- Menyiapkan struktur repository
- Menyusun dokumentasi awal
- Menentukan sistem kerja project
- Menjadikan GitHub sebagai Source of Truth utama

## Struktur Repository

```txt
client/
server/
docs/
README.md
FEATURES.md
```

### client

Folder untuk frontend website.

### server

Folder untuk backend, API, dan logic server.

### docs

Folder untuk dokumentasi project, workflow, catatan, dan panduan kerja.

### FEATURES.md

Dokumen daftar fitur website Kosuka Bali Trip.

## Prinsip Project
- GitHub adalah Source of Truth utama.
- Perubahan harus dilakukan secara bertahap.
- Setiap tahap kerja harus jelas, kecil, dan mudah dicek.
- Dokumentasi harus dijaga agar project bisa dilanjutkan walaupun akun, room chat, atau AI tools berubah.
