# Kosuka Bali Trip (KBT)

**Kosuka Bali Trip** adalah sebuah website travel agency yang menawarkan kemudahan pemesanan paket wisata, transportasi, dan berbagai aktivitas liburan menarik di Bali.

## Tujuan Project
Menyediakan platform digital yang informatif, menarik, dan mudah digunakan bagi calon wisatawan untuk mencari, melihat detail, dan melakukan pemesanan layanan tur di Bali secara online.
