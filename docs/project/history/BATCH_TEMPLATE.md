# Batch Entry Template

Gunakan format ini untuk mencatat batch di dalam file history.

Satu file history berisi 10 batch.

Contoh:
- `BATCH_000_TO_010.md`
- `BATCH_011_TO_020.md`
- `BATCH_021_TO_030.md`

---

## Batch XXX — Batch Name

Status: Planned / In Progress / Completed / Needs Review / Blocked

Commit hash:
```txt
Pending
```

Date:
```txt
YYYY-MM-DD
```

### Objective

Tuliskan tujuan utama batch ini.

### Scope

Tuliskan batasan pekerjaan batch ini.

Contoh:
- Membuat file tertentu
- Mengubah dokumentasi tertentu
- Menyiapkan struktur tertentu

### Files Created

Tuliskan file yang dibuat.

Jika tidak ada, tulis:
- None

### Files Updated

Tuliskan file yang diubah.

Jika tidak ada, tulis:
- None

### Result

Tuliskan hasil pekerjaan.

### Review Notes

Tuliskan catatan review dari Room Chat 01.

Jika belum direview, tulis:
- Pending review

### Next Step

Tuliskan langkah berikutnya setelah batch ini selesai.

### Notes

Tuliskan catatan tambahan jika ada.
