# Project Context

## Project Name

Kosuka Bali Trip

## Project Type

Website travel dan tour berbasis Bali.

## Tujuan Project

Project ini bertujuan membuat website profesional untuk layanan travel dan tour di Bali.

Website ini akan digunakan untuk:
- Memperkenalkan brand Kosuka Bali Trip
- Menampilkan paket tour
- Menjelaskan layanan travel
- Mengarahkan calon customer untuk bertanya atau booking
- Menjadi identitas digital utama Kosuka Bali Trip

## Kondisi Project Saat Ini

Project sedang dimulai dari nol.

Tahap awal project adalah:
- Membuat struktur folder
- Menyiapkan dokumentasi dasar
- Menentukan sistem kerja project
- Menjadikan GitHub sebagai Source of Truth

## Source of Truth

GitHub repository adalah Source of Truth utama project.

Artinya:
- Struktur project yang benar adalah yang ada di GitHub
- File final adalah file yang sudah masuk repository
- Commit hash digunakan sebagai bukti perubahan
- ChatGPT, Gemini, dan Antigravity tidak menjadi sumber kebenaran final

## Prinsip Kerja Project

Project dikerjakan secara bertahap.

Setiap perubahan harus:
- Jelas tujuannya
- Kecil scope-nya
- Mudah dicek
- Tidak mencampur banyak pekerjaan sekaligus
- Sesuai instruksi yang diberikan

## Catatan Penting

Jangan langsung membangun fitur besar sebelum dokumentasi dan struktur dasar siap.

Jangan membuat asumsi besar tentang teknologi, framework, desain, database, atau deployment sebelum ada keputusan dari user melalui Room Chat 00.
