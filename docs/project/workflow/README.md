# Project Workflow Documentation

Folder ini berisi dokumentasi sistem kerja project Kosuka Bali Trip.

Dokumen di folder ini dibuat agar project tetap bisa dilanjutkan dengan konsisten meskipun akun ChatGPT, room chat, atau AI tools yang digunakan berubah.

## Dokumen

### PROJECT_CONTEXT.md

Berisi konteks utama project.

Dokumen ini menjelaskan:
- Project ini apa
- Tujuan project
- Cara memandang repository
- Prinsip kerja utama
- Hal yang harus dipahami sebelum membantu project

### WORKING_SYSTEM.md

Berisi sistem kerja project.

Dokumen ini menjelaskan:
- Peran user
- Peran Room Chat 00
- Peran Room Chat 01
- Peran Antigravity Agent
- Peran Gemini 3.1 Pro Low
- Peran Gemini 3.1 Pro High
- Alur instruksi, eksekusi, commit, push, dan review

### MODEL_USAGE_GUIDE.md

Berisi panduan penggunaan model AI dalam project.

Dokumen ini menjelaskan:
- Kapan menggunakan Gemini 3.1 Pro Low
- Kapan menggunakan Gemini 3.1 Pro High
- Kapan boleh menggunakan model alternatif
- Aturan Acceleration Mode
- Peran Room Chat 00 dan Room Chat 01 dalam rekomendasi model

## Prinsip Utama

GitHub adalah Source of Truth utama.

ChatGPT Project, Gemini, dan Antigravity Agent hanya menjadi alat bantu kerja. Semua keputusan dan status final project harus mengacu pada repository GitHub.
