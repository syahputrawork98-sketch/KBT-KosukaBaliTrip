# MVP Scope (Minimum Viable Product)

## Scope MVP
Fokus utama MVP adalah menghasilkan leads dan menampilkannya dengan baik.
1. **Landing Page**: Banner promosi, paket unggulan, footer.
2. **Katalog Paket Wisata**: List semua paket lengkap dengan thumbnail dan harga mulai dari.
3. **Detail Halaman Paket**: Menampilkan deskripsi, itinerary, fasilitas (include/exclude).
4. **Booking Flow (Simple)**: Pengunjung menekan tombol "Pesan Sekarang" yang mengarahkan ke form data singkat, lalu di-redirect ke WhatsApp beserta format pesanan yang terisi otomatis.
5. **Halaman Info**: About Us dan Contact.

## Non-MVP / Ditunda
Fitur berikut tidak akan dikerjakan pada rilis fase 1 (MVP):
- Integrasi dengan bank atau payment gateway.
- Sistem manajemen akun pengguna (Login/Register untuk user biasa).
- CMS Backend yang dinamis dan kompleks (Data untuk sementara dapat di-hardcode atau menggunakan database sederhana/headless CMS).
- Fitur keranjang belanja (Cart system multi-item).

## Batasan Project Awal
- **Fokus Utama**: Konversi prospek/leads ke WhatsApp.
- **Performa**: Harus ringan dan responsive di perangkat mobile (Mobile-first approach).
